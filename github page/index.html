ABOUT GAME

เกมSnake ladder มีจุดประสงค์จัดทำขึ้นเพื่อสร้างบันเทิงให้กับผู้เล่น โดยเกมนี้เป็นแนวเกมบันไดงู จำนวนผู้เล่น 2 - 4 คน
ซึ่งพวกเราได้ใช้ภาษาซีและgame library (raylib) ในการพัฒนาและพวกเราได้ออกแบบให้เกมบันไดงูนี้มีความแตกต่างไปจากเกมบันไดงูในรูปแบบเดิม
โดยเราเพิ่มฉากต่อสู้รูปแบบturn based, ระบบไอเทม,ระบบHP สอดแทรกเข้าไปในเกมบันไดงูนี้ด้วย
เพื่อเพิ่มความสนุกสนานในการเล่น และ เพิ่มความน่าสนใจในกับเกมนี้ด้วย ซึ่งผู้ชนะเกมนี้คือผู้ที่สามารถไปถึงเส้นชัยได้คนแรก!!!
